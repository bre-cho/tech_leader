# Executive Summary

Mục tiêu: biến MVP từ công cụ tạo ảnh đơn lẻ thành **AI Design-to-Video Operating Environment** có nhiều agent phối hợp khép kín.

## Luồng chính
Customer Input → Business Diagnosis Agent → Industry Adaptation Agent → Image Design Agent → Image QA Agent → Upsell Opportunity Agent → Video Concept Agent → Storyboard Agent → Offer Agent → CRM Follow-up Agent → Analytics Agent → Memory Update → Technical Lead Audit Agent.

## Build ưu tiên
### Phase 1 — Revenue Entry MVP
1. Design Studio
2. Image Generator
3. Image Scoring
4. Upsell Analyzer
5. Video Concept Preview

### Phase 2 — Upsell Conversion Engine
1. Storyboard Generator
2. Video Package Offer
3. CRM Follow-up
4. Upsell Dashboard

### Phase 3 — Closed-loop Automation
1. Auto video render
2. Memory learning
3. Industry-specific playbooks
4. Winner DNA recall

## Release Gate
GO khi: API contract ổn, agent output có schema, workflow trace có lineage, score lưu được DB, upsell recommendation xuất hiện đúng điều kiện, winner DNA được lưu sau selection/conversion.
